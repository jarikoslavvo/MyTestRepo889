# Начинаю изучение Git

## Git - распределенная система контроля версий , которая позволяет разработчику , или команде разработчика полностью контролировать репозиторий, а также
## связывать свой локальный репозиторий с сервисом хостинга репозиториев - GitHub. 

# Почему стоит использовать Гит? 

## * - система контроля версий , откат в случае необходимости
## * - возможность разработки новых функций приложения путем создания новых веток приложения , которые не влияют на основную ветку.
## * - командная работа всех разработчиков проекта с единым централизованным репозиторием на ГитХаб.
